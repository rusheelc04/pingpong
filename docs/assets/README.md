# Asset Conventions

This directory keeps screenshots and diagram exports in one place.

- `screenshots/` holds current product captures that can be embedded in the README.
- `diagrams/` holds static diagram exports that are easy to skim quickly.
- `docs/diagrams/` holds the editable Mermaid source for architecture diagrams.

Naming rules:

- Use lowercase file names with hyphens.
- Name screenshots after the page or state they show, for example `landing-page.png` or `live-match.png`.
- Keep one obvious "hero" image that can sit near the top of the README.
